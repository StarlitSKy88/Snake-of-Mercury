---

## 等级体系

```
序列 9 (最低) → 序列 8 → 序列 7 → 序列 6 → 序列 5 → 序列 4 → 序列 3 → 序列 2 → 序列 1 → 序列 0 (真神) → 序列之上 (旧日/伟大存在)
                      |←── 低序列 →→|←──── 中序列 ────→|←────────── 高序列（半神）──────────→|← 真神 →|← 旧日 →|
```

| 等级 | 说明 | 对应 |
|------|------|------|
| 序列 9-8 | 低序列 | 能力初显，可被普通火器击败 |
| 序列 7-5 | 中序列 | 能力增强，仪式变得关键 |
| 序列 4-3 | 圣者 | 漫长寿命，部分神话生物形态 |
| 序列 2-1 | 天使 | 完整神话形态，可回应全球祈祷 |
| 序列 0 | 真神 | 可改写途径内所有低序列的能力 |
| 序列之上 | 旧日/伟大存在 | 与源质融合，超越真神 |
